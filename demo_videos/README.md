This folder contains demonstration videos of the **Personalized Networking Assistant** project.

- **File:** `Personalized_Networking_Assistant_Demo.mp4`
- **Duration:** ~15 minutes
- **Description:** Complete live demonstration of the project including:
  - Problem statement and solution overview
  - System architecture and AI models (BART & GPT-2)

## Features Demonstrated

1. **Event Analysis & Theme Extraction** - BART-based zero-shot classification
2. **Conversation Starter Generation** - GPT-2 powered personalized message generation
3. **Real-time Fact-Checking** - Wikipedia API integration for claim verification
4. **Interactive UI** - Multi-page Streamlit application with feedback widgets
5. **Analytics Dashboard** - Historical data visualization and trend analysis
6. **Testing & Quality Metrics** - 38 automated pytest cases with >85% code coverage
